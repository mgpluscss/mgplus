Badges

Combine the mg-badge class with other classes to create different types of badges. You can use the mg-badge class to create a basic badge, and then add additional classes to change its appearance. The following classes are available: mg-badge danger, mg-badge success, mg-badge warning. You can also use the mg-text-s, mg-text-m, mg-text-l, and mg-text-xl classes to change the text size.
