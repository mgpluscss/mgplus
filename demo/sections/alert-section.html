Alert

mg-alert

Combine the mg-alert class with other classes to create different types of alerts. You can use the mg-alert class to create a basic alert box, and then add additional classes to change its appearance. The following classes are available: mg-alert danger, mg-alert success, mg-alert warning. You can also add an icon to the alert box by using the mg-icon class. The icon can be closed by using the icon-close class.
